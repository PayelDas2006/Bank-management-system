# 📚 Smart Library Management System (C + DSA)

## 📖 Overview

The **Smart Library Management System** is a console-based application
developed in **C Programming** using **Data Structures and Algorithms
(DSA)**. It provides an efficient way to manage library operations
through a simple, user-friendly, menu-driven interface.

## ✨ Features

-   Add Book
-   Display Books
-   Search Book
-   Update Book Details
-   Delete Book
-   Issue Book
-   Return Book
-   Sort Books
-   File Handling
-   Menu-Driven Interface

## 🧠 DSA Concepts Used

-   Arrays
-   Structures
-   Linked List
-   Stack
-   Queue
-   Linear Search
-   Binary Search
-   Bubble Sort
-   File Handling

## 🛠️ Technologies Used

-   C Programming
-   GCC Compiler
-   File Handling
